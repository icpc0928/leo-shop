# Leo Shop 🛍️

現代化電商購物網站，使用 Next.js 16 + TypeScript + Tailwind CSS 建構。

## 技術棧

- **框架：** Next.js 16 (App Router)
- **語言：** TypeScript
- **樣式：** Tailwind CSS v4
- **狀態管理：** Zustand（規劃中）

## 開發

```bash
npm run dev     # 啟動開發伺服器 http://localhost:3000
npm run build   # 建構生產版本
npm run start   # 啟動生產伺服器
```

## 頁面

| 路徑 | 說明 |
|------|------|
| `/` | 首頁 |
| `/products` | 商品列表 |
| `/products/[slug]` | 商品詳情 |
| `/cart` | 購物車 |
| `/about` | 關於我們 |
| `/contact` | 聯絡我們 |
| `/faq` | 常見問題 |

## 設計風格

簡約風格，參考 LeArts Shopify 主題。主色調金色 `#c8956c`，大量留白，乾淨排版。
